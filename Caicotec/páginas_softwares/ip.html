Protocolo de Internet (IP)

O que é IP

O IP, ou traduzindo, protocolo de internet é um protocolo de rede que é responsavél por identificar um host, computador, em uma rede mundial ou local. Devido ter a função de identificação, em todo o mundo, com milhares de máquinas ligadas a todo o tempo, é impossível que exista mais de um IP igual na rede mundial de computadores, a Internet.

Falamos em nosso site sobre as interfaces de um computador. Um PC possui uma interface específica que recebe um IP, que é a interface de rede, ou como mais conhecida a placa de rede do computador. É através desta que o computador se torna possível receber um endereço IP e se conectar na rede mundial.

Classes de IP

Os endereços de IP não são números quaisquer. Os endereços de IP são formados por quatro octetos que têm o intervalo de : 0.0.0.0 até 255.255.255.255, que em binário fica da seguinte forma: 0.0.0.0 até 11111111.11111111.11111111.11111111. Além desta regra os endereços são dividos em Classe A, Classe B, Classe C. Também existem a Classe D e Classe E, mas ainda não estão em pleno funcionamento, portanto iremos tomar mais ênfase nas três primeiras classes.

Classes de Endereços
Classe Intervalo
A 0.0.0.0 - 127.255.255.255
B 128.0.0.0 - 191.255.255.255
C 192.0.0.0 - 223.255.255.255

Endereços válidos e possíveis

Endereços possíveis e válidos
Classe Possíveis Válidos
A 0.0.0.0 - 127.255.255.255 1.0.0.1 - 126.255.255.254
B 128.0.0.0 - 191.255.255.255 128.0.0.1 - 191.255.255.254
A 192.0.0.0 - 223.255.255.255 192.0.0.1 - 223.255.255.254

A diferença de endereços possíveis e válidos de uma classe de endereço é a seguinte. Os possíveis é todo o intervalo de endereços de uma classe, como a tabela mostrada anteriormente. Os endereços válidos são os IPs que é possível atribuir à interface de rede para que a mesma conecte o computador na rede. Por exemplo se um endereço da classe fosse atribuído com o seguinte valor: 10.255.255.255, este não seria válido pois é um endereço de Broadcast ou 10.0.0.0, que seria um o endereço de rede.
