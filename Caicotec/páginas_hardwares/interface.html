Interface

O que são interfaces

Imaginamos que uma placa mãe é uma grande empresa com diversos funcionários. Os funcionários irão exercer determinadas funções de acordo sua especialidade, agora podemos comparar as interfaces com estes funcionários. Basicamente cada "tipo" de interface de um computador desempenham determinadas funções que vão de exibição de imagens na tela do monitor até a conversão de informações binárias em dados de rede de internet. Um exemplo bem simples é a interface de disco rígido, que nas placas mãe mais modernas são utilizadas as interface SATA (Serial ATA) ou a interface de rede. Em geral as interfaces são circuitos que enviam dados de outros dispositivos da placa mãe para a CPU processar.

Onboard e Offboard

Onboard quer dizer sobre a tábua, no caso na placa mãe e diferentemente da onboard, existem as offboard, ou fora da placa mãe. Apesar de já existir interfaces integradas na placa mãe com um bom desempenho, muitas pessoas ainda costumam adquirir interface offboard. Iniciantes podem achar estranho isto, mas na verdade as placas offboard geralmente apresentam maior desempenho do que as onboard. Um bom exemplo são as placas de vídeo, apesar de já haver uma placa integrada na placa mãe, muitas pessoas comprar uma placa offboard por questão de desempenho, já que a integrada não oferece um desempenho necessário que o usuário deseja, como por exemplo para jogos ou mesmo para uma pessoa que trabalha com edição gráfica. Há também a placa de wireless, no qual possibilita que um computador capture sinais de WiFi.
